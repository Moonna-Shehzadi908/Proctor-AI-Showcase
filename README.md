
# 🛡️ Proctor AI – AI-Based Online Exam Proctoring System

🏆 **Award-Winning Final Year Project** 

Proctor AI is an AI-powered online examination proctoring system developed to improve the security, fairness, and integrity of online assessments through intelligent real-time monitoring.

The project combines Artificial Intelligence, Computer Vision, and Full-Stack Web Development to detect suspicious activities during online examinations, providing a smarter and more reliable online testing experience.

## ✨ Key Features

* AI-based online exam monitoring
* Real-time face detection
* Object detection
* Suspicious activity detection
* Secure user authentication
* Student and Admin dashboards
* Responsive web interface
* REST API integration
* Supabase database integration
